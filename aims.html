Project Aims

The SIDEMIX project investigates the stiffness degradation and micromechanical behaviour of sand–fine mixtures under cyclic loading. The main objectives are:

  • Characterize physical–mechanical properties of sands and sand–fine mixtures.
  • Determine the minimum and maximum void ratios of tested materials.
  • Quantify particle morphology using Scanning Electron Microscopy (SEM).
  • Determine static behaviour through drained triaxial tests to define the critical state line.
  • Analyse cyclic behaviour using dynamic triaxial and torsional shear tests.
  • Develop and integrate Discrete Element Method (DEM) simulations for micromechanical analysis.
